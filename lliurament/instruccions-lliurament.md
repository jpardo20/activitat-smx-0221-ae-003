# Instruccions de lliurament (GitHub Classroom)

1. Accepta l’**Assignment** al Classroom i crea el teu repo privat.
2. Clona el repo i copia-hi aquest contingut de `activitat-smx-0221-ae-003`.
3. Crea la teva fitxa a partir de `/templates`:
   - **Opció MD:** duplica `plantilla-fitxa-torre.md` → `fitxa-torre.md` i omple.
   - **Opció HTML:** duplica `plantilla-fitxa-torre.html` → `fitxa-torre.html` i omple.
4. Desa les **fotos** a `fotos/` i les **evidències** a `evidencies/` (scripts a `/scripts/*`).
5. Executa (opcional) les eines d’auto-check:
   ```bash
   python tools/validate_no_os.py
   python tools/check_placeholders.py
   ```
6. Fes `commit` i `push`. Comprova que el repo conté:
   - `fitxa-torre.md` **o** `fitxa-torre.html`
   - carpeta `fotos/` (amb 5–8 imatges)
   - carpeta `evidencies/` (amb sortides de maquinari)
7. Envia l’enllaç del repo al Classroom.

> **Recorda:** Cap dada de sistema operatiu al text o a les captures.
